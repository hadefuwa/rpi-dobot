‚ñ∂ How the Vision System Works

The Vision System automatically detects and counts circular objects (counters) on colored backgrounds. It uses optimized circle detection algorithms to reliably find round objects.

🎥 Camera Connection

  • Connect a USB camera to the Raspberry Pi
  • The camera feed starts automatically when the page loads
  • The system automatically detects the first available USB camera (index 0)
  • Camera resolution is set to 640√ó480 pixels at 20 FPS
  • Live stream is displayed in MJPEG format for efficient transmission

🔍 Circle Detection Method

üéØ Circle Detection

The system uses HoughCircles algorithm with optimized preprocessing to reliably detect circular shapes. It applies Gaussian blur and adaptive thresholding to enhance circle edges before detection.

This method works best for detecting round counters on colored backgrounds (like blue conveyor belts).

⚙️ How It Works

  1. Image Capture: The system continuously captures frames from the USB camera
  2. Preprocessing: Converts to grayscale, applies Gaussian blur, and adaptive thresholding to enhance circle edges
  3. Circle Detection: Uses HoughCircles algorithm with optimized parameters to find circular shapes
  4. Filtering: Filters detections by size and confidence to remove false positives
  5. Display: Shows the count and draws bounding boxes around detected counters on the analyzed image

📊 Results Display

  • Objects Detected: Shows the total number of circular counters found in the current frame
  • Last Updated: Shows when the detection was last performed
  • Analyzed Image: Shows the camera frame with green bounding boxes and labels around detected counters

💡 Tips for Best Results

  • Ensure good, even lighting so circles are clearly visible
  • Use a colored background (blue works best) to contrast with the circular counters
  • Keep the camera at a consistent distance from the objects
  • Make sure the circular objects are clearly visible and not overlapping
  • The system automatically runs detection every second - no manual controls needed
  • Check the analyzed image to see which objects were detected

🔧 Technical Details

  • Library: OpenCV (cv2) for image processing
  • Detection Method: HoughCircles algorithm with optimized preprocessing for reliable circle detection
  • Frame Rate: 20 FPS for live stream, detection runs every 1 second
  • Resolution: 640√ó480 pixels
  • Stream Format: MJPEG (Motion JPEG) for efficient web streaming
  • Processing: Automatic - no manual controls needed
